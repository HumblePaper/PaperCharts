graph.set_margin(top, bottom, left, text)

this function sets the margin for the graph in the canvas.

Kind: static method of graph

Param Type Description
top int : corresponds to the top margin
bottom int : corresponds to the bottom margin
left int : corresponds to the left margin
text int : corresponds to the right margin

graph.set_canvas(width, height)

this function sets the width and heigth of the canvas.

Kind: static method of graph

Param Type Description
width int : corresponds to the width of the canvas
height int : corresponds to the height of the canvas

graph.bubbleChart(data, xAxis, yAxis, r, text)

this is the bubblechart function plots a static bubble chart with a constant radius bubble chart

Kind: static method of graph

Param Type Description
data json : json data obtained from the ajax code in example function
xAxis string : the json key which corresponds to the value plotted in the xAxis
yAxis string : the json key which corresponds to the value plotted in the yAxis
r string : the json key which corresponds to the radius of the bubbles plotted
text string : the json key which corresponds to the text inside the bubbles plotted

graph.heatMap(data, area, color, name)

this function plots a heat map using the above given parameters,

Kind: static method of graph

Param Type Description
data json : the json object obtained after passing the json file to ajax code in the example
area string : the json key which corresponds to the area of the cell in the heatmap
color string : the json key which correspond to the color depth of the cell
name string : the json key which corresponds to the cell name(i.e. text inside the cell)

graph.networkGraph(data, startup_name_key, startup_investor_key, startup_id_key, investor_name_key, investor_investments_key, investor_id_key)

this function plots a network of nodes of startups and their investors

Kind: static method of graph

Param Type Description
data json : the json object obtained after passing the json file to ajax code in the example
startup_name_key string : the json key which corresponds to the name of the startup node
startup_investor_key string : the json key which corresponds to the investors of the startup node
startup_id_key string : the json key which corresponds to the unique id of the startup node
investor_name_key string : the json key which corresponds to the name of the investor node
investor_investments_key string : the json key which corresponds to the investments of the investor node
investor_id_key string : the json key which corresponds to the unique id of the investor node
